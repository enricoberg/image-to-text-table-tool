## About The Project


The application aims to solve the issue of extracting data from scanned documents, in particular tables.
The tool allows you draw custom windows on the screen, matching the format of your table.
Once aligned with the table, you can convert the image to text and save it to a file of your choice, with the desired formatting.

### Usage Tutorial
Watch this short video to learn how to use this tool.
 
[![IMAGE ALT TEXT HERE](https://i.postimg.cc/J7Y3F8Jn/preview.png)](https://youtu.be/uIbXo3T0bwg)
In short: 
- CTRL+G and drag the mouse to draw rectangles
- CTL+ENTER to reset rectangles
- CTRL+U to save the current snapshot as text
- ESC to close the application
  
(You can change the shortcuts to match your needs)

## Getting Started



### Installation
First, get Python 3 installed  and clone this repository.
https://www.python.org/downloads/

Then install all the required packages:
```
pip install -r requirements.txt
```

You also need to have Tesseract installed. You can find the Windows direct download link  [here.](https://github.com/UB-Mannheim/tesseract/wiki)

Install it and make sure the executable is saved at the following path:
```xml

C:\Program Files\Tesseract-OCR\tesseract.exe

```
If your installation path is different, you can change the path in the app.py script.

  
  

<!-- USAGE EXAMPLES -->

You're good to go! Launch the app.py  script and have fun!
